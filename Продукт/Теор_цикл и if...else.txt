Циклы в Java и операторы.
	Цикл While работает до тех пор, пока условие станет верным.
Цикл будет выполняться снова и снова, пока переменная (i) меньше 5:

int i = 0;
while (i < 5) {
  System.out.println(i);
  i++;
}

	Цикл do/while это разновидность варианта while. Этот цикл выполнит блок кода один раз, прежде чем проверять, истинно ли условие, затем он будет повторять цикл, пока условие истинно.

do {
  // блок кода, который нужно выполнить
}
while (условие);


	Цикл for используется, если вы точно знаете, сколько раз должен выполниться цикл.

for (statement 1; statement 2; statement 3) {
  // блок кода, который нужно выполнить
}

Оператор 1 выполняется (один раз) перед выполнением блока кода.
Оператор 2 определяет условие выполнения блока кода.
Оператор 3 выполняется (каждый раз) после выполнения блока кода.


	Цикл for-each используется для перебора элементов массива:

for (type variableName : arrayName) {
  // блок кода, который нужно выполнить
}


Операторы.
	В Java есть след. условные операторы:
if для указания блока кода, который будет выполняться, если указанное условие истинно.
else для указания блока кода, который будет выполняться, если то же условие ложно.
else if чтобы указать новое условие для проверки, если первое условие ложно.
switch для указания множества альтернативных блоков кода для выполнения.
 
	Оператор if используется, чтобы указать блок кода, который должен быть выполнен, если условие true.

if (условие) {
  // блок кода, который должен быть выполнен, если условие истинно
}

	Оператор else используется, чтобы указать блок кода, который должен быть выполнен, если условие ложно:

if (условие) {
  // блок кода, который должен быть выполнен, если условие истинно
} else {
  // лок кода, который должен быть выполнен, если условие ложно
}

	Оператор else if используется для указания нового условия, если первое условие ложно.

if (условие1){
  // блок кода, который должен быть выполнен, если условие1 истинно
} else if (условие2) {
  // блок кода, который должен быть выполнен, если условие1 ложно и условие2 истина
} else {
  // блок кода, который должен быть выполнен, если условие1 и условие2 ложно
}
