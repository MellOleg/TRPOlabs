Классы и объекты.
Классы и объекты - это два основных аспекта объектно-ориентированного программирования. Все в Java связано с классами и объектами, а также с их атрибутами и методами. Например: в реальной жизни автомобиль - это объект. У автомобиля есть атрибуты , такие как вес и цвет, и методы , такие как вождение и торможение.
	Чтобы создать класс, используется ключевое слово class:

public class Main {
  int x = 5;
}

Создание объекта.
	Чтобы создать объект Main, укажите имя класса, за которым следует имя объекта, и используйте ключевое слово new:

public class Main {
  int x = 5;

  public static void main(String[] args) {
    Main myObj = new Main();
    System.out.println(myObj.x);
  }
}

Использование нескольких классов.
	Вы также можете создать объект класса и получить к нему доступ в другом классе. Это часто используется для лучшей организации классов (один класс имеет все атрибуты и методы, а другой класс содержит main()метод. Помните, что имя java-файла должно совпадать с именем класса.
	Когда оба файла скомпилированы:
C:\Users\Your Name>javac Main.java
C:\Users\Your Name>javac Second.java
