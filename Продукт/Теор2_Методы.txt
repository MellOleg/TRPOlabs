Методы.
Метод представляет собой блок кода , который работает только тогда , когда он вызывается. Они используются для выполнения действий называемых функциями. Методы следует использовать для повторного использования кода: определите код один раз и используйте его много раз.

Для создания метода его следует объявить в классе.

public class Main {
  static void Method() {
    // код, который должен быть выполнен
  }
}

Для вызова метода, напишите имя метода с двумя круглыми скобками () и точкой с запятой ;

Вызов Method внутри main:

public class Main {
  static void myMethod() {
    System.out.println("I just got executed!");
  }

  public static void main(String[] args) {
    myMethod();
  }
}
