Массивы Java.
	Массивы используются для хранения нескольких значений в одной переменной вместо объявления отдельных переменных для каждого значения. Чтобы объявить массив, укажите тип переменной в квадратных скобках :

String [] cars;

	Чтобы заполнить массив строк, вы можете написать:

String[] cars = {"Car1", "Car2", "Car3"};

	Чтобы создать массив целых чисел, вы можете написать: 

int[] num = {10, 20, 30, 40};

	Вы можете получить доступ к каждому элементу массива, обращаясь к нему по его номеру. Важно знать, элементы массива начинаются с 0, а не с 1

String[] cars = {"Car1", "Car2", "Car3"};
System.out.println(cars[0]);
// выведет Car1

	Если вы хотите изменить элемент массива, то вы можете сделать это след. образом:

Cars[0] = "Car4";

	Чтобы узнать длинну массива, используется свойство length:

	System.out.println(cars.length);

	Как уже говорилось ранее, чтобы перебирать элементы массива используется цикл for или for-each:

String[] cars = {"Car1", "Car2", "Car3"};
for (int i = 0; i < cars.length; i++) {
  System.out.println(cars[i]);
}
