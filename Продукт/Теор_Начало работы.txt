Добро пожаловать в приложенеи по изучению Java. Для начала обучения необходимо иметь возможность работы с Java. Для успешного начала, ниже приведен способ установки Java на ОС Windows.

Возможно у вас уже установлена Java, если вы не знаете, то это можно проверить. Нажмите на кнопку "Windows" в ней введите cmd и нажмите Enter. Вы увидете консоль, в ней нужно написать: 

C:\Users\ваше имя*>java -version

после этого, если у вас есть Java будет выведено примерно:

java version "11.0.1" 2018-10-16 LTS
Java(TM) SE Runtime Environment 18.9 (build 11.0.1+13-LTS)
Java HotSpot(TM) 64-Bit Server VM 18.9 (build 11.0.1+13-LTS, mixed mode)

Если же у вас этого не оказалось, то вы можете скачать её с сайта oracle.com.
Что бы установить Java требуется:
	1. Перейдите в «Свойства системы» (можно найти в Панели управления> Система и безопасность> Система> Расширенные настройки системы).
	2. Нажмите кнопку «Переменные среды» на вкладке «Дополнительно».
	3. Затем выберите переменную «Путь» в Системных переменных и нажмите кнопку «Изменить».
	4. Нажмите кнопку «Создать» и добавьте путь, по которому установлена ??Java, а затем введите \ bin . По умолчанию Java устанавливается в C: \ Program Files \ Java \ jdk-11.0.1 (если при установке ничего не было указано). В этом случае вам нужно будет добавить новый путь: C: \ Program Files \ Java \ jdk-11.0.1 \ bin.
Затем нажмите «ОК» и сохраните настройки.
	5.Наконец, откройте командную строку (cmd.exe) и введите java -version, чтобы узнать, запущена ли Java на вашем компьютере.

Каждое приложение в Java начинается с имени класса, и он должен соответствовать имени файла.
Файл можно создать в любом текстовом редакторе. Попробуйте создать его с именем Hello и написать в нем следующее:

public class Hello{
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Сохраните код в Блокноте как «Hello.java». Откройте командную строку (cmd.exe), перейдите в каталог, в котором вы сохранили файл, и введите «javac Main.java»:

C:\Users\ваше имя*>java Hello

Результат: Hello World
