Переменные в Java - контейнеры для хранения данных.
Существуют разные типы переменных, например:
	String - хранит в себе текст и заключается в двойные кавычки.
	Int - хранит ТОЛЬКО целые числа.
	Float - может хранить числа с плавающей запятой
	Char - может хранить отдельные СИМВОЛЫ, значения которых заключены в одинарные кавычки.
	Boolean - сохраняет значения с 2-мя состояниями True или False.

Что бы объявить переменную, необходимо указать тип и присвоить ему значение:

	тип имя* = значение;

Так же можно объявить переменную без присвоения ей значения и присвоить его позже:

int num;
num = 1;
System.out.println(num);

Результат:1

С помощью метода println() можно использовать для добавления переменной к другой переменной:

String firstName = "Oleg ";
String lastName = "Mell";
String fullName = firstName + lastName;
System.out.println(fullName);

или для объединения текста и переменной:

String name = "Oleg";
System.out.println("Hello " + name);

так же можно использовать для числовых занчений в методе, оператор + для получения их суммы:

int x = 2;
int y = 3;
System.out.println(x + y); 



Типы даннх делятся на 2 груаппы:
	Примитивные - byte, short, int, long, double,float,char, boolean.
	Не примитивные - string, arrays, classes.
Примитивные типы определяют тип и размер значений и не имеет дополнительных методов.

Приведение типов.
Приведение типов - присвоение значения одного примитивного типа данных другому. Существует 2 типа преобразований:
	Неявное (автоматически) преобразование меньшего типа в больший:

public class Main {
  public static void main(String[] args) {
    int Int = 9;
    double Double = Int; 

    System.out.println(Int);      
    System.out.println(Double);   
  }
}

	Явное (вручную) преобразование большего размера в меньший:

public class Main {
  public static void main(String[] args) {
    double Double = 9.78d;
    int Int = (int) Double; 

    System.out.println(Double);   
    System.out.println(Int);      
  }
}
