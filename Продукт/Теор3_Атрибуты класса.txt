Атрибуты класса.
	Можно сказать, что атрибуты класса - это переменные внутри класса. Создайте класс с именем " Main" с двумя атрибутами: x и y:

public class Main {
  int x = 5;
  int y = 3;
}

Вы можете получить доступ к атрибутам, создав объект класса и используя точечный синтаксис (.). Создайте объект с именем " myObj" и распечатайте значение x:

public class Main {
  int x = 5;

  public static void main(String[] args) {
    Main myObj = new Main();
    System.out.println(myObj.x);
  }
}

Вы также можете изменить значения атрибутов:

public class Main {
  int x;

  public static void main(String[] args) {
    Main myObj = new Main();
    myObj.x = 40;
    System.out.println(myObj.x);
  }
}

	Если вы создаете несколько объектов одного класса, вы можете изменять значения атрибутов в одном объекте, не влияя на значения атрибутов в другом:

public class Main {
  int x = 5;

  public static void main(String[] args) {
    Main myObj1 = new Main();  // Object 1
    Main myObj2 = new Main();  // Object 2
    myObj2.x = 25;
    System.out.println(myObj1.x);  // Outputs 5
    System.out.println(myObj2.x);  // Outputs 25
  }
}
